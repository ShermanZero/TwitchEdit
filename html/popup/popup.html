TwitchEdit v1.6.0
This is a very simple program. TwitchEdit adds an EDIT button to your clips in the Twitch Clip Manager page.
It works for both Clips I've Created and Clips of My Channel.
Why do you need this?
Well if anybody presses the clip button on your stream, but doesn't actually publish the clip, the clip is saved. However, to edit this mysterious clip, you have to open it up via the clip page, and then append '/edit' to the URL to actually edit (including trim), the clip.
THIS BUTTON DOES THOSE THINGS AUTOMATICALLY FOR YOU
